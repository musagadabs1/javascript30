Event Handling and Recursion

This project demonstrate how event handling works. On the side, you will see those cool boxes built with recursion. This project aligns with Wes's JS30 Project 25. Clicking BOX 1 will trigger an event handler, starting on from it's chain of outermost parent "BOX 2", then moves inwards. I use {capture: true} setting to achieve this behavior. A click on BOX 2 will trigger an event that fires only once, bubbling does not take place. Feel free to tweak JS code, try different cases to get better grasp on the concept.
