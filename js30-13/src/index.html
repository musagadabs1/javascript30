Arrays

To clone an array, use following methods.

      let arr = [1,2,3,4,5];
      let arrClone = arr.slice();
    
      let arr = [1,2,3,4,5];
      let arrClone = [].concat(arr);
    
      let arr = [1,2,3,4,5];
      let arrClone = [...arr]; // BETTER
    
      let arr = [1,2,3,4,5];
      let arrClone = Array.from(arr);

Objects

To clone an object, use following methods.

      let obj = { age: 12 };
      let objClone = Object.assign({}, obj, { age: 18 });
    
For deep level cloning, use related function available in lodash.
      let obj = { age: 12, hobbies: { indoor: ['a', 'b'], outdoor: ['c', 'd'] } };
      let objClone = JSON.parse(JSON.stringify(obj));
      objClone.hobbies.indoor[0] = 'e';
    
      let obj = { age: 12, hobbies: { indoor: ['a', 'b'], outdoor: ['c', 'd'] } };
      let objClone = { ...obj };
      objClone.age = 14;
      objClone.hobbies.indoor[0] = 'e'; // DOES NOT WORK IN DEEP LEVEL
